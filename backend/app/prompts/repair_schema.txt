{
  "labels": ["string", ...],
  "datasets": [
    {
      "label": "string",
      "data": [number, ...]
    }
  ]
}

RULES:
1. "labels" is a required array of strings. Infer from context if missing.
2. "datasets" is a required array. Each item needs "label" (string) and "data" (array of numbers).

5. Numeric strings like "42" should be converted to numbers.
6. If the input is scatter data (x/y pairs), convert each x value to labels and y values to data.
7. If the chart type is "pie" or "doughnut", keep only the first dataset.
8. Never invent data. If a required value cannot be inferred, use 0.
9. Produce only valid JSON. No trailing commas. No comments.

After the JSON, on a new line starting with "CHANGES:", list each transformation you made
as a comma-separated list of short phrases.
